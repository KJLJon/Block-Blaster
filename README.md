# Block Blaster

A web-based puzzle game inspired by the popular mobile game Block Blast. Place block pieces on the 8x8 grid to complete rows and columns.

## How to Play

- **Drag** block pieces from the tray at the bottom onto the 8x8 grid
- Complete full **rows or columns** to clear them and earn points
- Clear multiple lines at once for **combo bonuses**
- Clear lines on consecutive placements for **streak bonuses**
- The game ends when no remaining pieces can be placed

## Features

- 8x8 grid with 30+ unique piece shapes
- 3 pieces dealt per round
- Drag-and-drop with ghost preview (mouse and touch)
- Row and column clearing with animations
- Combo and streak scoring system
- High score saved to localStorage
- Responsive design for mobile and desktop
- Haptic feedback on supported devices

## Play

Open `index.html` in a browser, or visit the GitHub Pages deployment.
